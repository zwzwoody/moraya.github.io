# 好你个moraya

hexo介绍

Hexo 是一个快速、简洁且高效的静态博客框架。它使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

#### 核心特性

* **超快速度**：Node.js 驱动，海量文章瞬间生成

* **一键部署**：支持 GitHub Pages, Netlify, Vercel 等平台

* **丰富的插件系统**：轻松扩展功能

* **强大的主题系统**：数百款主题自由选择，支持自定义

* **Markdown 支持**：专注写作，无需关注排版

#### 工作流程

1. 创建新文章：`hexo new "文章标题"`
2. 编写内容：使用 Markdown 语法
3. 生成静态文件：`hexo generate`
4. 本地预览：`hexo server`
5. 部署上线：`hexo deploy`

#### 为什么选择 Hexo？

* 完全免费且开源

* 对开发者友好，高度可定制

* 无需数据库，托管成本低

* 社区活跃，问题容易解决

#### 快速开始

```bash
# 安装 Hexo
npm install -g hexo-cli

# 初始化博客
hexo init blog
cd blog

# 安装依赖
npm install

# 启动本地服务器
hexo server
```

现在访问 <http://localhost:4000> 就能看到你的博客啦！

#### 进阶技巧

* 使用 `_config.yml` 配置文件自定义博客设置

* 通过主题配置文件调整外观和布局

* 利用 Front-matter 管理文章元数据

* 搭配 Travis CI 或 GitHub Actions 实现自动部署

#### 注意事项

* 确保 Node.js 版本在 10.13 或以上

* 部署前记得配置部署信息

* 定期备份源文件到 GitHub 仓库

* 更新前请阅读版本变更说明
